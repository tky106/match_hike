# README
# アプリケーション名
  「match hike」
# 概要
  同一の目的地に向かう人同士のマッチングを提供するアプリケーション。同乗者を募集するホストの投稿に対してユーザーが応募する仕組み。、
# 本番環境
  http://54.150.247.16/
# 制作背景
  お金がない大学生の帰省や地方在住大学生の就活をサポートするため。また社会人が車で出張を行う際などに伴う余剰スペースの有効活用のため。
# 工夫したポイント
  ・ユーザーが応募したあと、同乗者のことを互いに知る機会を与えるためにトークグループを設けた。
